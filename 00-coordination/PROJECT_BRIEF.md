# Project Brief

## Project

simple-dev-portfolio

## User Idea

Build a simple one-page portfolio website for a software developer.

## Required Sections

- Hero section with name, title, and short intro
- About section
- Skills section
- Projects section with 3 sample projects
- Contact section

## Goal

Create a clean, professional, one-page portfolio website that presents a software developer's identity, background, skills, selected work, and contact options.

## Workflow

This project will move through research, build, QA, and fix loops until QA approves it or identifies a blocker.
